# Chart System

Charts should explain movement, confidence, and context.

## Chart Types

- ranking movement
- confidence over time
- prediction history
- market heat
- roster forecast
- scholarship balance
- source volume
- map legends

## Rules

- Always label axes or provide direct labels.
- Include confidence where relevant.
- Show "mock", "projected", or "simulated" labels when applicable.
- Provide Explain This for non-obvious calculations.
- Never rely on color alone.
