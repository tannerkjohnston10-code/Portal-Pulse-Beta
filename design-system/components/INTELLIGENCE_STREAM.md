# Intelligence Stream

The Intelligence Stream is Portal Pulse's signature UI element: a thin, elegant ribbon that communicates live movement without becoming noisy.

## Content Types

- verified updates
- prediction changes
- visits
- commitments
- market movement
- AI insights
- watchlist events

## Required Elements

- event type
- entity label
- confidence indicator
- source count badge
- timestamp
- expandable details

## Interactions

- smooth scrolling
- pause on hover
- keyboard focus pause
- filters by sport, confidence, team, watchlist, event type
- expandable details drawer or popover

## Motion Rules

- New items slide in subtly.
- Important changes pulse once, not continuously.
- Reduced motion disables scrolling animation and uses static pagination.

## Trust Rules

- Prediction items must say projection.
- Simulated items must say simulated.
- Conflicting items must show conflict state.
- Source count must be visible.
