# Map System

Maps are intelligence surfaces, not decorative backgrounds.

## Map Uses

- transfer movement
- recruiting pipelines
- regional activity
- conference movement
- NIL geography
- coaching footprint
- historical replay

## Rules

- Every layer needs a legend.
- Every flow needs confidence and source count.
- Hover and click states should reveal evidence and context.
- Reduced motion should disable animated flows.
- Maps must remain readable on mobile.
