# AI Interface Guidelines

AI should feel useful, calm, and accountable.

## Rules

- AI never presents unsupported conclusions as facts.
- AI outputs should show confidence and evidence.
- AI panels should distinguish proposed, reviewed, approved, rejected, simulated, and projected states.
- AI thinking placeholders should name the operation, such as "checking evidence" or "comparing sources."
- AI should offer next actions, not trap users in chat-only workflows.

## Athena Panels

Athena panels should show briefing status, evidence state, confidence summary, and Core approval status.

## Agent Outputs

Agent cards should include agent name, role, proposed finding, confidence, evidence count, and review state.

## Explainability

Every AI insight must support Explain This.
