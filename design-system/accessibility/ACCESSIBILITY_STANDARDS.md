# Accessibility Standards

PP-DOS targets WCAG 2.2 AA.

## Keyboard Navigation

- All interactive controls must be reachable by keyboard.
- Focus order should match visual order.
- Drawers and modals should manage focus.

## Focus States

- Focus rings must be visible against dark and light themes.
- Icon-only buttons require accessible labels.

## Reduced Motion

- Honor reduced-motion preferences.
- Disable continuous scrolling and decorative motion.
- Preserve state changes through text and layout.

## Color Contrast

- Text and essential UI must meet WCAG 2.2 AA contrast.
- Status cannot rely on color alone.

## Screen Reader Support

- Use semantic HTML.
- Name controls clearly.
- Announce dynamic stream changes only when important and user-relevant.

## Responsive Behavior

- Touch targets should be at least 40px.
- Dense tables should scroll or transform into cards.
- Mobile navigation should remain understandable.
