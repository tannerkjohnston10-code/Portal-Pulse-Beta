# Typography System

Typography should communicate hierarchy, density, and trust.

## Roles

- **Display:** major product moments, founder pages, high-level workspace titles.
- **Workspace Headers:** dashboard/product workspace identity.
- **Dashboard Cards:** compact titles, numbers, labels, status summaries.
- **AI Summaries:** readable explanations, evidence summaries, briefings.
- **Analytics:** tabular numbers, deltas, scores, rankings.
- **Data Tables:** compact uppercase headers, readable rows.
- **Charts:** small labels, axes, tooltips, confidence notes.
- **Timelines:** readable sequence text with metadata.
- **Labels:** uppercase, compact, consistent.
- **Mobile:** deliberate type scale that preserves readability.

## Rules

- No viewport-based font scaling.
- Use tabular numerals for scores, rankings, and deltas.
- Keep letter spacing modest; avoid negative letter spacing.
- Dense dashboards should use compact type, not tiny type.
- Explanations should use comfortable line height.
