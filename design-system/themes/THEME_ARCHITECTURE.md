# Theme Architecture

PP-DOS themes are semantic variations, not separate brands.

## Dark

Primary theme. Dense, premium, intelligence-first.

## Light

Secondary theme for daylight reading, reports, and broader accessibility preferences.

## High Contrast

Accessibility-first theme with stronger borders, clearer focus, and reduced translucency.

## Coach Mode

Professional workflow theme for staff and roster tools. Slightly calmer, denser, and more utilitarian.

## Presentation Mode

Large-format theme for media, investor, meeting, and broadcast-ready displays.

## Rules

- Themes must preserve semantic meaning.
- Confidence colors remain consistent across themes.
- Theme changes should not alter data interpretation.
