# AI Intelligence Engine

AI modules are architecture-only in this phase. They define structured extraction, entity resolution, verification, confidence scoring, conflict detection, summarization, timeline intelligence, knowledge graph queries, prompt templates, and prediction signals.

Rules:

- AI never writes directly to trusted database facts.
- AI outputs claim candidates, verification packets, summaries, graph edges, and review recommendations.
- AI must not invent facts, commitments, NIL values, or sources.
- Every output should explain confidence and contradictions.

See `docs/AI/INTELLIGENCE_ENGINE.md`.
