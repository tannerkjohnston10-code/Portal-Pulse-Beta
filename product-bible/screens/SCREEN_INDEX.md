# Screen Index

This folder will contain screen-level product specifications.

Initial screen groups:

- Command Center
- Player Intelligence Center
- Team Intelligence Center
- Rankings
- TMX
- Athena
- Knowledge Graph
- Maps
- Replay
- Team Builder
- Developers
- Admin

Each screen spec should describe purpose, primary question, target users, layout, states, trust indicators, navigation, mobile behavior, and future enhancements.
